# Security Policy

## Reporting
Report security-sensitive issues privately through GitHub security advisories when available, or contact the repository owner directly.

## Supported Versions
The current supported version is the latest released version.

## Scope
AgentFrame is documentation-first. Security concerns usually involve unsafe instructions, workflow gaps, misleading validation, or guidance that could cause data loss, secret exposure, or unauthorized changes.

## Handling
Security fixes must preserve existing useful content, document the risk, include validation where possible, and update release notes.
