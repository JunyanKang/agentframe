# Project State Template

Use this directory to initialize `.codex/project/` in a repository that adopts AgentFrame's optional governance framework.

Rules:

- Copy these template files into the target repository's `.codex/project/` directory.
- Replace every `Unknown - requires human input` entry with confirmed facts from the target repository.
- Delete template instructions only after the corresponding real project fact is recorded.
- Do not copy AgentFrame's own `.codex/project/` files as target-project facts.
- Preserve existing target-repository instructions and record conflicts instead of overwriting them.
