# AgentFrame Starter Kit

Use this starter kit when a repository needs AgentFrame's optional project memory and governance state.

Copy `starter-kit/.codex/project/` into the target repository's `.codex/project/` directory, then replace every `Unknown - requires human input` entry with confirmed facts from that repository.

Do not copy AgentFrame's own `.codex/project/` directory into another project. That directory contains AgentFrame repository facts, not reusable starter content.
