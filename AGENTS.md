# AgentFrame Repository Instructions

- Read `.codex/AGENTS.md` before making changes.
- Preserve existing skills and framework files.
- Use `skills/agentframe-*` as the canonical installable Codex skills.
- Use `.codex/framework/` for reusable framework policy.
- Use `.codex/project/` only for project-local state.
- Do not delete, rename, or overwrite existing useful content without explicit approval.
- Work one task at a time.
- Run `npm run validate` before completion.
