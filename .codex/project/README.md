# AgentFrame Project State

This directory contains project-local facts for the AgentFrame repository itself.

Do not copy these files as target-project facts when adopting AgentFrame in another repository. They mention AgentFrame-specific architecture, releases, roadmap, risks, and memory. Copying them unchanged can mislead future Codex sessions into treating AgentFrame's repository state as the target project's state.

For a new adopter repository, start from `.codex/framework/project-template/` and replace every `Unknown - requires human input` entry with confirmed facts from the target repository.
